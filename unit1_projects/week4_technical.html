Arrays vs Hashes

Arrays

Array elements are accessed by indices. This is why arrays are good for look ups (searching for information). Arrays allow you to store information of the same datatype into a single variable

Hashes

In a hash table, you can set the values and keys to anything you like. You can search by strings, numbers or objects! That's a lot of freedom
