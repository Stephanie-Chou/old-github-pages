Why JavaScript?

Easy, Easy and More Easy

When you search "Why is JavaScript so popular?" you come across a myriad of answers. But the most common word you see is "Easy." Javascript is easy to learn, easy to use, easy to edit and easy to extend (jquery!).

Events

Javascript is used to make a user experience better. You can start to integrate more interesting effects on a webpage such as onClick and onHover events. According to css-tricks.com:

"JavaScript is the only language available for having your website communicate with the browser and watch for events: clicks, double clicks, mouse enters, mouse exits, key presses, browser window sizing... "

AJAX

HTML, CSS, JSON, XMLHttprequest, XML and DOM are collectively known as AJAX. They can all be used in conjunction with JavaScript with ease
