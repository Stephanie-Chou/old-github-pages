Vocabulary

  • Class: A way of organizing information and producing new objects. Ruby has some built in classes we're already familiar with such as Array.
  • Object: Objects are instances of a class.
  • Instance Variable: You can identify an instance variable because they are preceded with an '@' symbol. Instance variables are tied to the instance of a class.

    • Here is an example of using the instance variables and instance methods within one of this weeks challenges. Lets start with 1_die because that was a simpler challenge

    
class Die
	def initialize(sides)
		raise ArgumentError.new(" Must have at least 1 side") if sides [lessthan] 1
			@die_sides= sides
		end 
	end
		def sides
		@die_sides
		end
		def roll
		1+rand(@die_sides)
		end
end

    Here you can see that we only have one instance variable called @die_sides. This is a variable that can change with each instance of the class Die. For example, calling die=Die.new(6) will create the instance variable @die_sides=6. This @die_sides is a unique variable and value to die. If you delete die, then that instance of @die_sides will no longer exist.

    def sides and def roll are instance methods. Like instance variables, these methods are attached to the instance of the class. When you call sides or roll on our newly created die instance, you will get a unique output.
