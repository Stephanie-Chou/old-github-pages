What are the differences between relative, absolute, and fixed positioning

Relative

A relative object is positioned within the normal flow of the page. This means that they will not overlap and will be positioned relative to the other objects around them. The below rectangle and circle are relative

relative!

Absolute

Absolute objects are positioned relative to the parent container. Absolute positioned objects are often used within a relatively positioned object. An absolute object is out of the 'normal' flow of the page. Objects positioned within the normal flow of the page after an absolute object will overlap the absolute object. The below blue rectangle is absolute while the black circle is relative.

Fixed

A fixed position object will not move even if the page is scrolled. This is because the object is fixed relative to the browser window. The below rectangle is fixed
